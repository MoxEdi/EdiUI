## ğŸš€ Installation

### CurseForge
- Install `EdiUI` from [CurseForge](https://www.curseforge.com/wow/addons/ediui) or WowUP

### Manual
1. Download/extract this repository.
2. Place folder as: `World of Warcraft/_retail_/Interface/AddOns/EdiUI`
3. Restart WoW or run `/reload`

## âš™ï¸ Quick Start
- Install Elvui -> EdiUI Installer will open after reload.
or
- Open options: `/eui` or `/ediui`
- Open installer: `/eui install`
- Re-run installer: `/eui reinstall`
- Recalculate cluster: `/eui cluster`
- Reload UI: `/rl`
